# FIFA World Cup BI Dashboard

## Project Overview
This project is a Business Intelligence dashboard built using Power BI to analyze FIFA World Cup data, including team performance, goals, matches, and trends. It transforms raw data into interactive visual insights for better decision-making and analysis.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Features
- Team performance analysis  
- Goal statistics and match insights  
- Interactive dashboards and KPIs
